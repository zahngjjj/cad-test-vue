### blender怎么打开.DXF 的文件
Blender 默认是不能直接打开 .DXF 文件的，要用 “Import AutoCAD DXF” 插件（Add-on）来导入

菜单栏选择 Edit → Preferences…
打开扩展面板
搜索框输入 DXF
勾选 Import-Export: Autocad DXF Format (.dxf)
关闭 Preferences（设置会自动保存）
